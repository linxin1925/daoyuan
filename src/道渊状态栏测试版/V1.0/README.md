# 道渊状态栏 V1.0

## 文件
- `MVU状态栏-V1.0.json` — SillyTavern 正则导入文件（1.9MB）

## V1.0 功能清单

### 数据展示（MVU stat_data 全字段覆盖）
- **主角**：姓名(大)/境界(右小字)/灵根/宗门/五维/神念/状态
  - 二级菜单①：外貌穿搭（性别置顶 + 容貌/身形/衣着）
  - 二级菜单②：气运（点击查看详情）
  - 器灵头像（最右，点击随机台词，2 张图轮播）
- **道侣**：性别/种族/状态/境界/亲密/性格/外观/身高/背景/神通/心声 + 立绘
- **人物**：头衔/境界/好感/关系阶段/性格/描述 + 立绘
- **玉简**：联系人列表 + 微信式聊天（对方左/我右气泡）
- **绝色榜**：立绘缩略图 + 排名"第X名" + 仙姿 + 群芳谱（颜色区分）
- **储物袋/器物**：描述 + 数量
- **动向/机遇/功法/副业**：全字段展示

### 交互功能
1. **玉简传讯**（完整聊天）
   - 发消息 → 写入 MVU 变量 → AI 生成 NPC 回复 → 写回变量
   - 自定义 API 配置（OpenAI 兼容，留空用酒馆内建 generate）
   - 世界书知识注入（按联系人勾选）
   - 预设管理（保存/应用/删除）
   - 神识 ≥20 才能传讯
   - 删除单条消息（同步删变量）
2. **立绘系统**
   - 内置立绘库 200 条（玄天界 167 + 女性专用 33）
   - 自定义立绘（本地图片 base64 / URL）
   - 道侣/人物/绝色榜可查看立绘
3. **地图系统**
   - 玄天界（14 节点）+ 仙界（5 节点）
   - 节点连线、势力详情弹窗
   - 玄天界全图（点击放大）

## 导入方法
1. 酒馆角色卡《道渊》v5.2 → 正则脚本（局部正则）
2. 导入 `MVU状态栏-V1.0.json`
3. 启用正则（scriptName: MVU状态栏（测试））
4. AI 输出含 `<StatusPlaceHolderImpl/>` 占位符时自动渲染状态栏

## 依赖
- 酒馆助手扩展（JS-Slash-Runner）
- MVU 变量框架 + 《道渊》v5.2 角色卡
- 联网（jsdelivr CDN 加载 pinia + 立绘图床）
