## 关键架构

### webpack
- 入口自动 glob `src/**/index.ts`，道渊状态栏测试版有 index.ts 会被编译
- externals：jquery→`$`、lodash→`_`、vue→`Vue`、zod→`z`、toastr、yaml→`YAML`、gsap
- pinia、async-wait-until 走 jsdelivr CDN（`testingcf.jsdelivr.net`）
- 产物：`dist/道渊状态栏测试版/index.html`（1.9MB 单文件内联）

### 正则注入
- 角色卡局部正则 "MVU状态栏"（scope: character）
- find_regex: `<StatusPlaceHolderImpl/>`
- destination: `{display: true, prompt: false}`（markdownOnly: true）
- source: ai_output，runOnEdit: true
- replace_string 是 ` ```html ` 包裹的 dist 产物（**是 dist 快照，源码改后需重新同步**）

### store 数据流
- `store.ts` 用 `Mvu.getMvuData({type:'message', message_id: getCurrentMessageId()})` 读取
- 监听 `Mvu.events.VARIABLE_UPDATE_ENDED` 自动 refresh
- **关键**：手动 `Mvu.replaceMvuData` 写入**不会触发**该事件，需在写入后主动调 `store.refresh()`（已在 wxService/YujianPanel 处理）

## 关键决策与坑

1. **Transition 单根**：`<Transition mode="out-in">` 包裹的 v-if/v-else-if 链要求每个分支组件**单根节点**。YujianPanel 曾因加 WxSettingsModal 多根导致切换失效，改为放 PanelCard 内部解决。
2. **正则 destination**：导出 JSON 时 `markdownOnly` 必须为 `true`（对应 display:true），否则导入后状态栏不显示（曾导致云端导入不显示）。
3. **正则是 dist 快照**：源码改后需 `fetch dist + updateTavernRegexesWith` 重新同步正则，否则不生效。
4. **手动写变量不触发刷新**：`Mvu.replaceMvuData` 后需主动 `store.refresh()`。
5. **立绘图床**：`free-img.400040.xyz`（高清1024×1536）+ `i.postimg.cc`（部分缩略图120×180会模糊）。fetch 受 CORS 但 `<img>` 标签能加载。
6. **器灵 tab 已删**：器灵改为头像在主角栏最右，`$器灵台词` 变量仍存在并使用。
7. **气运 tab 已删**：气运改为主角姓名处二级菜单。
8. **图片裂开**：多为 URL 失效或网络瞬时，代码层有 onImgError 占位。

## 开发流程

### 改代码后同步正则
```bash
# 1. pnpm watch 自动编译（需在运行）
# 2. 本机 HTTP 服务器托管 dist（端口 8765）
cd dist && python3 -c "
import http.server, socketserver
class H(http.server.SimpleHTTPRequestHandler):
    def end_headers(self):
        self.send_header('Access-Control-Allow-Origin', '*')
        self.send_header('Cache-Control', 'no-store')
        super().end_headers()
print('serving dist on :8765', flush=True)
with socketserver.TCPServer(('', 8765), H) as httpd:
    httpd.serve_forever()
"
```

浏览器同步正则（chrome-devtools MCP）：
```js
const TH = window.TavernHelper;
const resp = await fetch('http://localhost:8765/%E9%81%93%E6%B8%8A%E7%8A%B6%E6%80%81%E6%B0%8F%E6%B5%8B%E7%89%88/index.html', { cache: 'no-store' });
const html = await resp.text();
await TH.updateTavernRegexesWith(
  (rs) => rs.map(r => r.script_name === 'MVU状态栏' ? { ...r, replace_string: '```html\n' + html + '\n```' } : r),
  { type: 'character', name: 'current' }
);
```

### 调试
- chrome-devtools MCP 连本机酒馆（pageId 2）或云端（pageId 1）

### API 速查
- 取变量：`TavernHelper.getVariables({type:'message'})`
- 取正则：`TavernHelper.getTavernRegexes({type:'character', name:'current'})`
- 改正则：`TavernHelper.updateTavernRegexesWith(updater, {type:'character', name:'current'})`
- Mvu 读写：`Mvu.getMvuData({type:'message', message_id})` / `Mvu.replaceMvuData(data, {...})`
- 取最后 AI 消息：`getChatMessages(-1, {role:'assistant'})`
- 世界书：`getCharWorldbookNames('current')` → `getWorldbook(name)`
- 生成：`generate({user_input})` 或自定义 fetch OpenAI 兼容
