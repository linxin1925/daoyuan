# 道渊状态栏 V1.2

> V1.1 基础上的增量更新，导出于 2026-07-05。

## V1.2 相对 V1.1 的变更

1. **玉简传讯完整功能**：发消息、AI 生成 NPC 回复、自定义 API 配置、世界书知识注入、传讯预设管理
2. **世界书分组折叠**：316 条按前缀自动分为 26 组，每组可折叠/展开/全选，搜索模式平铺
3. **浮动聊天窗口**：可拖拽的弹出式聊天窗口，联系人列表和聊天同时可见
4. **自动滚动**：发送/接收消息后自动滚到最新位置，点开聊天自动定位底部
5. **主角头像**：读取酒馆当前 persona 头像显示在姓名栏左侧
6. **标签栏自适应**：flex-wrap 自动换行 + 自动填满，不再横向滚动
7. **字体调整**：14px 基础字号
8. **UI 细节**：输入框返回按钮、消息气泡加宽、对齐和间距优化
9. **变量全覆盖**：补上了 动向.阶段、器物.等级、器物.损耗度 三个遗漏字段
10. **消息 ID 修复**：缓存 stat_data 消息 ID，AI 回复后不会丢失

## 导入方法
1. 酒馆角色卡《道渊》v5.2 → 正则脚本（局部正则）
2. 导入 `MVU状态栏-V1.2.json`
3. 启用正则

- 数据展示：主角（含器灵头像/气运二级菜单）/道侣/人物/玉简/绝色榜/储物袋/器物/动向/机遇/功法/副业
- 玉简传讯：发消息、AI 生成 NPC 回复、自定义 API、世界书注入、预设、删除消息、浮动窗口
- 立绘系统：内置 200 条 + 自定义（道侣/人物/绝色榜均可查看）
- 地图：玄天界（14 节点）+ 仙界（5 节点）+ 玄天界全图 + 势力详情
