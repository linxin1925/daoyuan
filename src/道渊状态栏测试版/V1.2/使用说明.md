# 道渊状态栏 V1.2 — 使用说明

> 《道渊》v5.2 角色卡专用 MVU 状态栏。通过正则替换注入 AI 回复楼层，以修仙画风面板展示角色数据，并支持玉简传讯（与 NPC 互发消息）。

## 一、安装

1. 酒馆中加载角色卡 **《道渊》v5.2**
2. 打开 **正则脚本** → **局部正则**（Character Regex）
3. 点击 **导入**，选择 `MVU状态栏-V1.2.json`
4. **启用**该正则

导入后，AI 每次输出 `<StatusPlaceHolderImpl/>` 时，酒馆会自动将其替换为状态栏面板。

## 二、面板概览

### 顶部信息栏
- 当前时间、当前地点、危机程度、遭遇冷却

### 主角面板
- **姓名**：点击展开二级菜单 → 性别 / 容貌 / 身形 / 衣着
- **头像**：读取酒馆当前用户 persona 头像，无头像显示占位图标
- **气运**：点击「气运」按钮展开 → 类型 / 效果 / 使用状态 / 剩余次数 / 压制状态
- **六维状态条**：生命 / 精血 / 灵力 / 修为 / 神识 / 道心
- **器灵**：最右侧圆形头像，点击随机切换头像和台词

| 标签 | 内容 |
|------|------|
| 功法 | 主角已习功法列表（类型/境界/熟练度/描述） |

| 机遇 | 任务/事件列表（难度/目标/机缘/引言） |

## 三、玉简传讯

### 基本使用
1. 切换到 **玉简** 标签
2. 点击联系人进入聊天 → 输入消息 → Enter 发送
3. AI 自动以该 NPC 身份生成回复

**神识限制**：神识 ≥ 20 才能传讯（原版机制）。

### 浮动窗口
- 聊天界面右上角点击 **↗ 弹出按钮**
- 窗口可**拖拽**（按住标题栏移动）
- 点击 **←** 返回联系人列表，点击 **×** 关闭
- 点击窗口外空白区域还原为嵌入模式

### 删除消息
- 每条消息右侧有垃圾桶图标，点击删除

### 玉简设定（齿轮按钮）

**传讯预设**：保存/应用/删除多套配置方案

**自定义 API 配置**（可选）：
- 基础 URL：OpenAI 兼容 API 地址
- API 密钥
- 模型名称
- 留空则使用酒馆内建生成

**角色知识注入（世界书）**：
- 按联系人独立勾选世界书条目
- 316 条自动按前缀分为 26 组
- 每组可**折叠/展开**、**全选/取消**
- 顶部工具栏：全选所有 / 取消全选 / 展开全部 / 折叠全部
- 搜索框：输入关键词平铺显示匹配条目
- 点击条目右侧 `>` 展开查看内容
- 已勾选条目以标签形式显示在搜索框上方

**传讯指引**：自定义提示词，影响 NPC 回复的语气/风格

## 四、立绘系统

- 内置 200 条立绘 + 支持自定义上传
- **道侣 / 人物 / 绝色榜** 左侧有缩略图，点击查看大图
- 无立绘时显示占位图标

## 五、地图

- **玄天界**：14 个节点（中央神州/北荒/南疆/东海/西漠 等），点击节点查看详情
- **仙界**：5 个节点
- 节点详情含势力列表，点击势力名弹出详细介绍
- **查看玄天界全图**：加载玄天界完整地图图片，点击可放大

## 六、注意事项

- 状态栏数据由 AI 输出的 `<UpdateVariable>` 块驱动更新，**每轮 AI 回复后自动刷新**
- 玉简消息存储在楼层变量中，切换聊天/刷新页面不会丢失
- 自定义 API 需兼容 OpenAI `/chat/completions` 格式
- 所有设置（API 配置/世界书勾选/预设）保存在浏览器 localStorage 中，清除浏览器数据会丢失
