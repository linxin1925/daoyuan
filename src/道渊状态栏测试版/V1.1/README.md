# 道渊状态栏 V1.1

> V1.0 基础上的增量更新，导出于 2026-07-05。

## V1.1 相对 V1.0 的变更
1. **人物面板**：每个 NPC 加立绘缩略图（最左，点击查看大图，无立绘时显示 person 图标占位）
2. **道侣面板**：每个道侣加立绘缩略图（同上，crane 占位）
3. **道侣 meta 移到名字右边**：性别·女 / 种族·人族 / 身高·168cm 从单独一行移到名字同一行右侧
4. **状态栏高度降低**：全局字号 15→13px，整体高度从 1010px 降至 673px（减少约三分之一）
5. **灵宠标签**：始终显示（原空对象时不显示，现 `show: true`）

## 导入方法
1. 酒馆角色卡《道渊》v5.2 → 正则脚本（局部正则）
2. 导入 `MVU状态栏-V1.1.json`
3. 启用正则
